# 🛰️ NIX-RAY 
### [ SYSTEM // HARDENED // MANAGER ]

NIX-RAY is a high-performance system utility designed to audit and purge **APT**, **Snap**, and **PIP** units through a unified, emerald-tinted dashboard. Built for speed and security.

## ⚡ Quick Start
1. **Clone:** `git clone https://github.com/rugerobuilds/nix-ray.git`
2. **Ignite:** `python3 launch.py`

## 🛡️ Security
- **Handshake Protocol:** 32-character session tokens.
- **Local-Only:** API binds strictly to `127.0.0.1`.
- **Atomic Operations:** Clean extraction via secure subprocess handling.
